<h1>LumoraBot</h1>
<hr>
<h3>Lumora is an english twitch bot who manage your custom commands and moderation.</h3>
<hr>

<h2>Default Commands</h2>

## Core Module
- !ping<br>
## Custom Commands Module
- !command add [command] [response]<br>
- !command remove [command]<br>
- !command edit [command] [response]<br>
- !command alias [command] [add/remove] [alias]<br>
## Stats Module
- !watchtime<br>
- !messages<br>
- !add-minutes [user] [amount]<br>
- !add-messages [user] [amount]<br>
- !remove-minutes [user] [amount]<br>
- !remove-messages [user] [amount]<br>
- !reset-minutes [user]<br>
- !reset-messages [user]<br>
- !block [user] [all/chatstats/watchtime]<br>
- !unblock [user] [all/chatstats/watchtime]<br>
