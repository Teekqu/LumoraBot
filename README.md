<h1>LumoraBot</h1>
<hr>
<h3>Lumora is an english twitch bot who manage your custom commands and moderation.</h3>
<hr>

<h2>Default Commands</h2>
> Core Module
- !ping
> Custom Commands Module
- !command add <command> <response>
- !command remove <command>
- !command edit <command> <response>
- !command alias <command> <add/remove> <alias>
> Stats Module
- !watchtime
- !messages
- !add-minutes <user> <amount>
- !add-messages <user> <amount>
- !remove-minutes <user> <amount>
- !remove-messages <user> <amount>
- !reset-minutes <user>
- !reset-messages <user>
- !block <user> <all/chatstats/watchtime>
- !unblock <user> <all/chatstats/watchtime>
